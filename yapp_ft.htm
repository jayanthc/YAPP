yapp_ft

NAME

yapp_ft - perform Fourier transform and optionally polyphase filtering on baseband data

SYNOPSIS

yapp_ft [options] baseband-data-file

DESCRIPTION

Performs Fourier transform and optionally polyphase filtering on baseband data. In other words, this is a software spectrometer. The input should be 8-bit, complex, dual-polarization data. The output is written in the headerless SIGPROC '.fil' format.

OPTIONS

-h, --help
Display a short help text.
-s, --skip-time time
Length of data to be skipped, in seconds (default is 0).
-p, --proc-time time
Length of data to be processed, in seconds (default is all data).
-b, --pfb
Turn polyphase filtering on.
-n, --nfft N
Number of points in FFT (default is 1024).
-a, --nacc acc-len
Number of spectra to be accumulated (default is 1 - no accumulation).
-l, --tsamp tsamp
Sampling time in seconds.
-f, --fil
Output spectra in filterbank format (currently the only format supported).
-e, --site site-name
Observing site name.
-r, --pulsar psr-name
Pulsar name.
-g, --graphics
Turn on plotting.
-i, --invert
Invert background and foreground colours in plots.
-n, --non-interactive
Run in non-interactive mode.
-v, --version
Display the version.

FILES

If running with PFB on, this program requires a filter coefficients file to be present in the current directory. Filter coefficients files can be created using the yapp_gencoeff.py script.

EXAMPLE

yapp_ft -b -n 4096 -a 10 -l 0.00000078125 -f -e GBT -r J1921+2153 rawdata.dat
