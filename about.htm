about

Yet Another Pulsar Processor (YAPP) is a radio pulsar data analysis software suite. The tools in this software suite include those for viewing data and associated metadata, converting baseband data to filterbank format, dedispersion, smoothing, filtering, coherent addition of multiple time series, folding, subtraction of time series, single-pulse-sifting, stacking sub-band dedispersed time series to form filterbank format data, along with scripts for calibrating and displaying folded profiles and computing spectral indices, and a few format-conversion utilities.

YAPP is very much in development, and the number of features it currently supports is limited. It is mainly intended to be used in conjunction with other pulsar data processing software such as SIGPROC or PRESTO, among others.
