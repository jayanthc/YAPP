YAPP
Yet Another Pulsar Processor
 

YAPP is a radio pulsar data analysis software suite. It is a work in progress, with the tools released so far listed below.

yapp_viewmetadata
Prints metadata/data header/configuration information to the standard output stream.
yapp_viewdata
Plots raw dynamic spectrum data to a PGPLOT device.

To get YAPP, either use git from the command line ('git clone git://github.com/jayanthc/yapp.git'), or go to the downloads page for a gzipped tarball or zip file.

System requirements: Linux/MAC OS, PGPLOT
