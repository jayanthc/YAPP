YAPP
Yet Another Pulsar Processor
 

Yet Another Pulsar Processor is a radio pulsar data analysis software suite. It is a work in progress, with the tools released so far listed below.

viewmetadata
Prints metadata/data header/configuration information to standard output.
viewdata
Plots raw dynamic spectrum data to PGPLOT device.

Created by Jayanth Chennamangalam with guidance from Avinash Deshpande. Acknowledgments to Yogesh Maan, Isaac Backus, and Duncan Lorimer who contributed to this work in varying degrees.
