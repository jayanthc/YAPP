yapp
yet another pulsar processor

Yet Another Pulsar Processor (YAPP) is a radio pulsar data analysis software suite. The tools currently released as part of this software suite include those for viewing filterbank data, viewing associated metadata, converting baseband data to filterbank format, and dedispersion.

Created by Jayanth Chennamangalam.
