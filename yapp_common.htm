common options

Given here are the options that are common to most YAPP tools.

Standard usage:

yapp_tool-name [options] data-file

The options are:

  • -h or --help: Display this usage information
  • -s or --skip time: Length of data to be skipped, in seconds (default is 0 s). This option is valid for all tools except yapp_viewmetadata.
  • -p or --proc time: Length of data to be processed, in seconds (default is all data). This option is valid for all tools except yapp_viewmetadata.
  • -n or --nsamp samples: Number of time samples read in one block (default is 4096 samples). This option is valid for all tools except yapp_viewmetadata and yapp_ft.
  • -g or --graphics: Turn on plotting. This options is valid for all tools except yapp_viewmetadata and yapp_viewdata.
  • -m or --colour-map name: Colour map for plotting. Supports some of the standard MATLAB colour maps, plus a few more. Valid colour map names are given below (default is 'jet'). This option is valid for all tools except yapp_viewmetadata.
    • 'autumn'
    • 'blue'
    • 'bone'
    • 'cool'
    • 'copper'
    • 'gray'
    • 'gray-inv'
    • 'hot'
    • 'jet'
    • 'pink'
    • 'spring'
    • 'summer'
    • 'winter'
  • -i or --invert: Invert background and foreground colours in plots (default is white on black). This option is valid for all tools except yapp_viewmetadata.
  • -e or --non-interactive: Run in non-interactive mode. This option is valid for all tools except yapp_viewmetadata.
  • -v or --version: Display the version.
